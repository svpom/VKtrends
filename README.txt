Приложение на maven и spring mvc. Все зависимости есть в мавене. Мавен собирать с аргументом ком.строки package.
На выходе war. Класть его в webapps tomcat'a. Tomcat запускать через startup.bat.
3 контроллера, которые доступны по http://localhost:8080/VKtrends/hello, http://localhost:8080/VKtrends/index и,
если перейти по сабмиту формы с index, то http://localhost:8080/VKtrends/get-posts?пары-ключ-значение-гет-запроса.

Теперь если ввести что-то во второе поле index'a, то эти данные отобразятся на get-posts. Возможно, прямая ссылка
на get-posts пока не сработает из-за необходимости(?) получения пар ключ-значение в гет-запросе по этмоу адресу.
Еще добавил в зависимости javax.servlet.
